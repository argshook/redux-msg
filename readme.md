# Redux Msg

small set of functions to help manage redux.

opinionated and experimental

example usage in [this repo](https://github.com/argshook/how-to-redux)

1. `npm i redux-msg`
1. 
    ```js
    import {
        create-message,
        create-messages-reducer,
        merge-reducers,

        create-selectors,
        create-reducer
    } from 'redux-msg';
    ```
